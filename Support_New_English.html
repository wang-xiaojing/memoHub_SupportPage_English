New mode is the screen for creating new memos. You can input various types of content including text, images, videos, audio recordings, location information, and tags to create rich memos.

üì± Screen Layout

Header Area

  • Cancel Button - Cancel memo creation and return to previous screen

  • Date Display - Current date and time

  • Text Editor - Input memo title and content
  • Media Display - Added images and videos
  • Audio List - Recorded audio files
  • Tag Display - Assigned tags

  • Location Button - Add current location
  • Tag Button - Add and manage tags

üíæ Save and Cancel Operations

Save Button

1Save Process

  • Tap the "Save" button in the top right corner
  • Memo data is saved to Core Data
  • Automatically returns to the previous screen
  • New memo appears in the list view

üíæ Auto-Save Function

  • All input content is automatically saved when Save is tapped
  • Text, media, location, tags are all preserved
  • Creation date and time are automatically recorded

Cancel Button

1Cancel Operation

  • Tap the "Cancel" button in the top left corner
  • All input content will be discarded
  • No data is saved to Core Data
  • Returns to the previous screen

‚ö†Ô∏è Data Loss Warning

  • No Recovery - Cancelled content cannot be recovered
  • No Confirmation - No confirmation dialog is shown
  • Immediate Effect - Cancellation occurs immediately

üìä Content Limits and Performance

Content Limitations

  • Media File Limits - Check app settings for maximum file counts
  • Storage Space - Large media files consume device storage
  • Performance Impact - Many media files may affect app performance

üí° Tips for Efficient Memo Creation

Best Practices

  • Descriptive Titles - Write clear first lines that serve as good titles
  • Organized Content - Use tags to categorize memos for easy searching
  • Media Quality - Consider file sizes when adding multiple media items
  • Location Privacy - Be mindful of location data when sharing memos
  • Regular Saving - Save frequently to avoid losing work

Workflow Suggestions

  1. Start with Text - Begin with a clear title and description
  2. Add Media - Include relevant photos, videos, or audio
  3. Set Location - Add location if relevant to the memo content
  4. Apply Tags - Categorize with appropriate tags
  5. Review and Save - Check content before saving

‚ö†Ô∏è Important Considerations

Data Management

  • No Auto-Save - Content is only saved when Save button is tapped
  • Permission Requirements - Camera, microphone, and location access require user permission
  • Device Storage - Media content is stored locally on the device
