List mode displays created memos in a list format and is the screen for viewing, editing, and deleting memos. It has two display modes: Standard display and Lite display, which can be switched according to your needs.

Standard Display Mode (Normal Mode)

  • Memo Title - Displayed with "Title" label
  • Memo Date - Creation date and time display
  • Alarm Information - Icon display when set
  • Text Content - Partial display of memo body text

Lite Display Mode (Simple Mode)

  • Compact Display - Hide "Title" label
  • Date Only - Display only basic information
  • No Text Display - Memo body text is not shown
  • Clean Appearance - Able to view more memos in list

1Tap the "Lite" button in the top right

2Display mode will switch

  • In Standard Mode - Button displays "Lite" with blue background
  • In Lite Mode - Button displays "Lite" with white background and blue border
  • Settings are automatically saved to Core Data

üîÑ Memo Operation Methods

Tap Operations

1Edit Memo

  • Tap any memo item in the list
  • Navigates to edit mode for that memo
  • All memo content becomes editable

Swipe Actions

1Delete Operation

  • Swipe left on a memo item
  • Red "Delete" button appears
  • Tap to permanently delete the memo

Delete Operation Caution

  • Permanent Deletion - Deleted memos cannot be recovered
  • Confirmation Dialog - No confirmation dialog is shown
  • Immediate Effect - Deletion occurs immediately upon tapping
